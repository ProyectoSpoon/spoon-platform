# Data Display Components

Este directorio contiene componentes para mostrar y visualizar datos de manera estructurada.

## 📋 Componentes Planificados

### Lists & Tables
- **SpoonList**: Listas verticales con items customizables
- **SpoonListItem**: Item individual de lista con avatars, acciones
- **SpoonTable**: Tablas responsivas con sorting y paginación
- **SpoonDataGrid**: Grillas de datos avanzadas

### Visual Indicators
- **SpoonAvatar**: Avatares de usuario con fallbacks
- **SpoonBadge**: Badges numéricos y de estado
- **SpoonTag**: Tags semánticas para categorización
- **SpoonProgressBar**: Barras de progreso lineales
- **SpoonProgressCircle**: Indicadores de progreso circulares

### Information Display
- **SpoonTooltip**: Tooltips informativos
- **SpoonPopover**: Popovers con contenido rico
- **SpoonAccordion**: Acordeones expandibles
- **SpoonTabs**: Sistema de pestañas

### Media
- **SpoonImage**: Imágenes responsivas con lazy loading
- **SpoonIcon**: Sistema de iconos del design system
- **SpoonEmptyState**: Estados vacíos con ilustraciones

## 🎯 Propósito

Los componentes de data display se enfocan en:

- **Legibilidad**: Presentar información de manera clara
- **Accesibilidad**: Soporte completo para screen readers
- **Performance**: Lazy loading y virtualización cuando es necesario
- **Responsividad**: Adaptación a diferentes tamaños de pantalla
